## 版本说明

### 版本格式

版本号格式为 `主版本.次版本.修订号`：

- **主版本**：不兼容的 API 修改
- **次版本**：向下兼容的功能性新增
- **修订号**：向下兼容的问题修正

### 变更类型

- **新增** - 新功能
- **变更** - 现有功能的变化
- **废弃** - 即将移除的功能
- **移除** - 已移除的功能
- **修复** - Bug 修复
- **安全** - 安全相关的修复

## 贡献指南

如果你想为此项目做贡献，请：

1. 在提交 Pull Request 前，确保更新此 CHANGELOG
2. 遵循 [Keep a Changelog](https://keepachangelog.com/) 格式
3. 在 `[未发布]` 部分添加你的更改
4. 发布新版本时，维护者会将更改移到对应的版本号下
